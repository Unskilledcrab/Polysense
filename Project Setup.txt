PROJECT SETUP
Right Click Solution (Polysense) -> Click "Set Startup Projects..."

Select "Multiple startup projects:" and use the settings below (In the correct order)

PS.Shared - None
PS.UI.Shared - None
PS.Web.API - Start
PS.UI.WPF - Start
